Master Full Stack Developer

April 2022 to December 2022

UNIR: Remote

Calificación final: 9.34

El master de Full Stack Developer está basado en la metodología MEAN. En él, hemos estudiado todas las tecnologías necesarias tanto para el desarrollo FrontEnd, como para el desarrollo BackEnd de esta metodología con base en JavaScript.

Como proyecto de fin de master se realizó una aplicación real para la empresa Inetum. La aplicación era una academía online con geolocalización de profesores en Angular y con conexión a una API REST propia creada con Node JS y con base de datos en MYSQL. Se trabajó en equipo (5 personas) con un repositorio en github y se utilizaron procesos para asemejar el proyecto a un proyecto real con ramas y pull request. Se siguió una metodología scrum.

Hemos realizado proyectos personales de FrontEnd como un carrito de la compra en Vanilla JS, la maquetación de una tienda online, un blog en Angular, un directorio de alumnos con conexión a una API REST.

Para la parte de BackEnd se han realizado proyectos como creación de una API REST con Node JS, Express JS y base de datos en SQl. Asimismo, una API REST con las mismas tecnologías pero con base de datos en MongoDB.

Las tecnologías aprendidas en el master fueron:

  • Angular
  • JavaScript
  • TypeScript
  • Node JS
  • Express JS
  • Mongo DB
  • MySQL
  • HTML
  • CSS
  • GIT
  • TDD
  • Firebase

Master Data Science

January 2020 to July 2021

UNED: Remote

Calificación final: 9.38

En el master de Data Science utilizamos como lenguajes de programación Python y R. Empezamos con conceptos clave de estadísitica, uso de herramientas para realizar un tratado óptimo de los datos(preprocesamiento), hasta ver los modelos más actuales (ARIMA, LR, RF, Prophet, XgBoost, Series, Random Forest...), hasta acabar con un módulo para estudiar la legalidad a la hora de tratar con datos.

Como proyecto de fin de master se realizó un estudio estadístico de ventas de un departamento de electrónica, utilizando ténicas de estimación propias de las series temporales. Dado que en el sector retail la previsión de compras es una de los pilares más importantes si se quiere tener éxito se hizo un estudio para conocer la demanda futura lo más acertada posible.

Se utilizó como herramienta cloud Google Colab y Azure. Utilizamos librerías de python para el desarrollo de los distintos gráficos.

Uno de los puntos clave para el desarrollo del master fue la utilización de GIT con repositorio en github y la creación de enviorments para los distintos estudios y así evitar la problemática de las versiones de las diferentes librerías.

Las tecnologías aprendidas en el master fueron:

  • Python
  • R
  • Estadística
  • Jupyter N
  • Algoritmos
  • Azure
  • AWS
  • Google Colab
  • Pre procesamiento
