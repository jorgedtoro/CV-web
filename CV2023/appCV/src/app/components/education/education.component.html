Master Full Stack Developer

April 2022 to December 2022

UNIR: Remote

Calificación final: 9.3

El master de Full Stack Developer esta basado en la metodología MEAN. En el hemos estudiado todas las tecnologías necesarias para el desarrollo FrontEnd como para el desarrollo BackEnd de esta metodología con base en JavaScript.

Como proyecto de fin de master se realizó una aplicación real para la empresa Inetum. La aplicación era una academía online con geolocalización de profesores en Angular y con conexión a una API REST propia creada con Node JS y con base de datos en MYSQL. Se trabajó en equipo (5 personas) con un repositorio en github y se utilizaron procesos para asemejar el proyecto a un proyecto real con ramas y pull request. Se siguió una metodología agile.

Hemos realizado proyectos personales de FrontEnd como un carrito de la compra en Vanilla JS, la maquetación de una tienda online, un blog en Angular, un directorio de alumnos con conexión a una API REST.

Para la parte de BackEnd se han realizado proyectos como creación de una API REST con Node JS, Express JS y base de datos en SQl. Asimismo, una API REST con las mismas tecnologías pero con base de datos en MongoDB.

Las tecnologías aprendidas en el master fueron:

  • Angular
  • JavaScript
  • TypeScript
  • Node JS
  • Express JS
  • Mongo DB
  • MySQL
  • HTML
  • CSS
  • GIT
  • TDD
  • Firebase
