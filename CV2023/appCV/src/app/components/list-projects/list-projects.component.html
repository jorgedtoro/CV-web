App Teacher

Aplicación de academia digital con geolocalización de profesores, alta y baja de usuarios, alta y baja de matrículas y perfiles privados.
Conexión con API REST propia en MYSQL.

Shopping Cart (RxJS)

Tienda con carrito de la compra. Se ha usado la libreria RxJS para toda la comunicación mediante observables entre los distintos componentes y para la respuesta de la API. La recuperación de los productos se gestiona mediante una llamada a una API pública.

Web Resume

Curriculum personal en Angular. Es una app responsive en la que se ha intentado cumplir con todas las buenas prácticas, intentando hacer un cógido sostenible en el tiempo.

ToDo with Firebase

Aplicación de tareas con conexión a Firebase. Se utilizan servicios de autenticación y posibilidad de iniciar con cuenta de google. La base de datos está alojada en Firestore, con almacenamiento en firebase Storage. En la aplicación se pueden crear tareas, listas de tareas y filtrar por ellas. Tiene un CRUD en firebase completo.

API REST MySQL

API REST con Express JS para una aplicación tipo blog de Angular con base de datos en MYSQL.

API REST MongoDB

Aplicación con Express JS con CRUD sobre grupo de datos que conforman los inmuebles de una finca, con conexión a API localhost en MongoDB. Para las acciones del modelo se ha usado Mongoose.

Angular App

Directorio de usuarios paginado con conexión a una API REST (peticiones.online) Maquetado con Bootstrap.

Carrito de la compra

Carrito de la compra funcional utilizando Vanilla JS y Bootstrap. Se usa POO para separar el carrito de los productos. Como API se generó un JSON en JSON Blob.
